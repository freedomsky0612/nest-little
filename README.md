## 介绍

项目启动成功首先需要满足以下两点：

1、你本机的mongodb数据库正常运行
2、你本机包含这个库“nest_little”（可以用自己的库，但需要在env文件配你这个库名）


## 安装

```bash
$ npm install
```

## 启动

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:test
```
